Web Teknolojileri

Web teknolojileri, internet üzerinde çalışan uygulamalar, siteler ve hizmetler geliştirmek için kullanılan araçlar ve standartların bir bütünüdür. İşte bu teknolojilere genel bir bakış:

1. Ön Yüz (Frontend) Teknolojileri

  • HTML (HyperText Markup Language): Web sayfalarının yapısını oluşturur.
  • CSS (Cascading Style Sheets): Web sayfalarının stil ve tasarımını belirler.
  • JavaScript: Dinamik ve etkileşimli içerikler oluşturmak için kullanılır.
  • Framework ve Kütüphaneler: React, Angular, Vue.js, jQuery gibi araçlar.

2. Arka Yüz (Backend) Teknolojileri

  • Programlama Dilleri: PHP, Python (Django, Flask), Ruby, Java, C#, Node.js.
  • Veritabanı Yönetim Sistemleri: MySQL, PostgreSQL, MongoDB, SQLite, Redis.
  • Web Sunucuları: Apache, Nginx, Microsoft IIS.


1. Ön Yüz (Frontend) Teknolojileri









HTML

Bir web sayfası tasarımında olmazsa olmazlardan ilki html (hyper text markup language) dir. Html dili (“”) küçüktür ve büyüktür işaretleri arasına eklenen etiket adı verilen İngilizce kısaltmalardan oluşan bir teknolojidir. Örneğin bir paragraf metni oluşturmak için


…paragraf metni…

yapısı kullanılır. Bu dil sayesinde internet sayfaları statik olarak tasarlanır. Html içerisine javascript, php, asp, vb. kodlar ile sayfalar daha etkileşimli hale getirilerek günümüz dinamik web sayfaları oluşturulmaktadır. Fakat işin temelinde ve olmazsa olmazı html’dir. HTML 4.01 versiyonunun ardından yaklaşık 20 yıl sonra gelen yeniliklerle HTML5.0 ile internet sayfaları ve web tarayıcılar da dönüşüme girmeye başlamıştır.








CSS

CSS (Cascading Style Sheets), web sayfalarının tasarımını ve düzenini kontrol etmek için kullanılan bir stil dilidir.

  • HTML ile oluşturulan içeriklerin renk, yazı tipi, arka plan, hizalama ve düzen gibi özelliklerini belirler.
  • Harici, dahili veya satır içi olarak uygulanabilir.
  • Web sayfalarında görsel tutarlılık sağlar ve bakımını kolaylaştırır.
  • Responsive tasarımlarla farklı cihazlarda uyumlu görünümler oluşturur.
  • tasarımının temel bileşenlerinden biridir.
Javascript

Buradaki script kelimesi “betik” anlamına gelmektedir. En yaygın yanlışlardan birisi de JAVA programlama dili ile javascriptin birbirine yakınmış gibi gösterilmesidir. Fakat bunları yaratan firmalar bile farklıdır. İsim benzerliğinden başka hiçbir benzerlik yoktur. Javascript “c” programlama dilinin webe dönük olarak kullanılmaya çalışılması ile ortaya çıkmıştır. Bu betik dili sayesinde web tarayıcınızda (browser) bulunan bütün nesnelere (yazılara, resimlere, katmanlara, formlara) hareketlilik verebilir ya da birbiri ile etkileşim kurabilecek işler yaptırabilirsiniz. Bu betik dili ile web sayfaları ilk zamanlardaki pasif halinden daha etkileşimli ve kullanıcılara daha renkli işler sunmamızı sağlamıştır.
JavaScript, web sayfalarını ilk zamanlardaki pasif halinden daha etkileşimli hale getirmiştir. Eskiden kullanıcılar sadece sayfa üzerinde sabit içerik görebilirken, JavaScript sayesinde artık sayfa içinde aktif bir şekilde etkileşimde bulunabiliyorlar. Kullanıcılar metinleri tıklayarak, formları doldurarak ya da resimlerin üzerine gelerek sayfa ile etkileşimde bulunabiliyorlar. Bu durum, web sayfalarının daha renkli, dinamik ve kullanıcı dostu hale gelmesini sağlamıştır.
Sonuç olarak, JavaScript, sadece web sayfalarındaki etkileşimi artırmakla kalmayıp, aynı zamanda web uygulamalarının hızla gelişen dinamik dünyasında önemli bir araç haline gelmiştir. Web tarayıcılarının temel yapı taşlarından biri olmayı başaran JavaScript, her geçen gün daha fazla özellik ve gelişmiş araçlarla kendini yenilemekte ve kullanıcıların daha etkileşimli ve dinamik deneyimler yaşamalarına olanak tanımaktadır.






Framework ve Kütüphaneler

React: Facebook tarafından geliştirilen, bileşen tabanlı ve sanal DOM kullanan bir JavaScript kütüphanesidir. Tek sayfalık uygulamalar için idealdir.

Angular: Google tarafından geliştirilen, MVC mimarisi ve çift yönlü veri bağlama desteği sunan bir JavaScript framework'üdür. Büyük projeler için uygundur.

Vue.js: Hafif, öğrenmesi kolay ve bileşen tabanlı bir framework'tür. Küçük ve orta ölçekli projelerde kullanılır.

jQuery: DOM manipülasyonu, olay işleme ve AJAX işlemleri için kullanılan, basit ve geniş uyumluluğa sahip bir JavaScript kütüphanesidir.


2. Arka Yüz (Backend) Teknolojileri

Programlama Dilleri

  • PHP: Web geliştirme için yaygın kullanılan, sunucu taraflı bir betik dilidir.
  • Python: Çok yönlü, kolay okunabilir bir programlama dili.
    • Django: Python tabanlı bir web çerçevesi; hızlı geliştirme ve temiz tasarımıyla bilinir.
    • Flask: Hafif, esnek bir Python web çerçevesi; minimal ve modülerdir.
  • Ruby: Temiz ve okunabilir bir sözdizimine sahip, nesne yönelimli bir programlama dili.
  • Java: Platformdan bağımsız, güçlü ve çok amaçlı bir dil.
  • C#: Microsoft tarafından geliştirilen, Windows uygulamaları için kullanılan bir dil.
  • Node.js: JavaScript'i sunucu tarafında çalıştırmayı sağlayan bir platform.

Veritabanı Yönetim Sistemleri

  • MySQL: Açık kaynaklı ve popüler bir ilişkisel veritabanı yönetim sistemi.
  • PostgreSQL: Güçlü, açık kaynaklı bir ilişkisel veritabanı.
  • MongoDB: Belge tabanlı (NoSQL) bir veritabanı; esnek veri modelleriyle ölçeklenebilir uygulamalarda kullanılır.
  • SQLite: Hafif, yerel dosya tabanlı bir veritabanı; küçük ölçekli projelerde idealdir.
  • Redis: Bellek içi çalışan, anahtar-değer depolama sistemi.

Web Sunucuları

  • Apache: Yaygın kullanılan açık kaynaklı bir web sunucusu.
  • Nginx: Yüksek performanslı ve ölçeklenebilir bir web sunucusu.
  • Microsoft IIS: Microsoft'un geliştirdiği, Windows platformunda çalışan güçlü bir web sunucusu.
